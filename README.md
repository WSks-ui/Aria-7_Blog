# Aria-7th Lab

Astro + native CSS/JavaScript static personal blog for Aria-7.

## Scripts

- `npm run dev` starts the local development server.
- `npm run check` runs Astro diagnostics.
- `npm run build` generates the static site into `dist/`.
- `npm run test:unit` runs unit tests and enforces coverage thresholds.
- `npm run test:e2e` builds the site and runs the Playwright suite.
- `npm test` runs the complete unit and end-to-end test suite.
- `npm run preview` previews the production build.

## Stack

- Astro content collections
- Native CSS and browser JavaScript
- Lucide Astro icons
